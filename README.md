# PresentDATA Theme

A customizable and minimal theme built for the [PresentDATA](https://github.com/mdxtend) platform.

## Supported Use Cases

This theme is designed to suit a variety of content types:

- **Academic** – Showcase your research, publications, and teaching work.
- **Blog** – Write and organize long-form or micro-blog content.
- **Photography** – Display curated photography portfolios with a clean layout.
- **Developer** – Share projects, code snippets, and technical documentation.
- **Articles** – Publish structured articles or essays with rich formatting.
